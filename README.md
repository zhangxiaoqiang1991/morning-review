# morning-review 晨间复盘

每天一次的结构化反思。AI 主动提问，帮你从工作日志中挖掘你没注意到的模式，提取今日优先级。

## 它用起来是什么感觉？

以下对话来自一次真实的晨间复盘（已脱敏）：

```
AI: 早上好。昨天你做了不少事——开发了新工具、写了复盘 skill、
    处理了一些飞书上的迁移工作。

    我观察到一个有意思的模式：你在情绪卡壳的时候，
    反而做出了当天最核心的产出。

    这让我想知道——**昨天有没有一个时刻，
    你发现原来自己想错了什么？**

你: 有的。我一直觉得情绪不好的时候什么都做不了，
    但昨天我发现情绪波动的时候，反而更适合做创造性的事。
    做工具、写东西的时候很容易进入心流，
    做完之后状态反而回来了。

AI: 所以你不是"等状态好了再行动"，
    而是"用行动修复状态"？

你: 对，就是这样。我从来没这么想过。

AI: 这个模式有名字——我叫它"修复型创作者"。
    大多数人的顺序：状态好 → 产出好
    你的顺序：状态不好 → 创造 → 状态变好。
    这个认知一旦被命名，你就可以主动用它了。

    而且我注意到另一件事——你昨天休息的时候选择了看脱口秀。
    **这是充电，还是逃避？**

你: 偏充电吧。看到差不多了，自己会有一个"够了"的信号。

AI: 好。那今天你的 Top 3 优先级是什么？

你: 第一，优化复盘 skill 并录个使用视频。
    第二，消化一个关于思辨能力的框架。
    第三，处理昨天剩下的迁移工作。

AI: 胜出条件是什么？

你: 如果今天把那套思辨框架消化并形成了自己的理解，
    这一天就是值的。

AI: 我要说的是——**你的最佳产出往往诞生于你最不在状态的时候。
    情绪不是你的敌人，它是创造力的前置条件。**
    今天如果情绪有小波动，别压它，直接去做需要创造力的事。
```

## 快速开始

```bash
git clone git@github.com:zhangxiaoqiang1991/morning-review.git
cp -r morning-review ~/.claude/skills/morning-review/
```

然后在 Claude Code 对话框中输入 `/morning-review` 启动。第一次运行会引导你完成配置。

## 四问复盘

| 问题 | 问什么 |
|------|--------|
| **认知反转** | 昨天有什么观点被颠覆了？ |
| **决策复盘** | 哪个决定最值得重看？ |
| **盲区发现** | 昨天忽略了什么？ |
| **能量信号** | 能量状态怎样？什么涨什么耗？ |

## 六维洞察透镜

AI 会从 6 个角度观察你的工作日志，找到你没注意到的模式：

| 透镜 | 它会看什么 |
|------|-----------|
| 模式显微镜 | 这个行为重复 10 次会形成什么？ |
| 决策基因解码 | 这个决策暴露了什么默认假设？ |
| 能量-产出关联 | 什么状态下产出最高？ |
| 身份张力诊断 | 你昨天切换了几个不同身份？ |
| 认知偏差命名 | 这个行为在心理学里叫什么？ |
| 反事实推演 | 如果做了不同选择会怎样？ |

## 完整工作流

```
Phase 0  加载工作日志
Phase 1  昨日速览（2 分钟）
Phase 1.5  AI 观察 + 六维洞察（3 分钟）
Phase 2  四问互动复盘（15 分钟）
Phase 3  认知种子 + 优先级提取（5 分钟）
Phase 4  误判心理学对照（可选）
Phase 5  保存并打开
```

## 数据源架构

复盘质量取决于输入数据的丰富度。核心是 Agent 工作日志（必需），辅助数据源用来覆盖更多工作轨迹。

```
Agent 工作日志（90% 核心）
  ├── 飞书日历（日程结构）
  ├── Get 笔记（语音灵感）
  ├── Screenpipe（屏幕活动）
  └── 微信读书（知识输入）
```

> 原则：**有则用，无则跳过。** 辅助维度非必需，只靠核心日志也能完成复盘。

## 底层逻辑

大多数复盘工具让你"填模板"，但复盘做不下去的原因不是没有模板，而是没有人戳破你的含糊其辞。

这个 skill 的核心差异：
- **洞察先行** — AI 从工作日志中读出你没注意到的模式
- **六维透镜** — 6 个观察维度拆解你的一天
- **三味燃料** — 每一层洞察都注入误判心理学（Munger 25 种）、跨学科思维模型、历史典故。帮你看到更大的坐标系
- **认知种子** — 每次复盘产出一颗可迁移的洞察
