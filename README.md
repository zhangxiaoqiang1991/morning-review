# morning-review 晨间复盘

每天一次的结构化反思。基于工作日志，通过 4 轮追问驱动认知复盘 + 今日优先级提取。

## 底层逻辑

大多数复盘工具让你"填模板"，但复盘做不下去的原因不是没有模板，而是没有人戳破你的含糊其辞。

这个 skill 的核心差异：
- **洞察先行** — 不只问问题，而是从你的工作日志中读出你没注意到的模式
- **六维透镜** — 从 6 个不同的观察维度拆解你的一天（模式显微镜、决策基因解码、能量关联、身份张力、认知偏差、反事实推演）
- **三味燃料** — 每一层洞察都注入误判心理学、跨学科思维模型、或历史典故，让你看到更大的坐标系
- **认知种子** — 每次复盘产出一颗可迁移的认知点，脱离今天场景依然成立

## 工作流程

```
Phase 0  加载工作日志（多数据源融合）
Phase 1  昨日速览（2 分钟）
Phase 1.5  AI 观察 + 六维透镜洞察
Phase 2  四问互动复盘（15 分钟）
Phase 3  认知种子 + 优先级提取（5 分钟）
Phase 4  误判心理学对照（可选）
Phase 5  保存到笔记并打开
```

## 四问复盘

1. **认知反转** — 昨天有什么观点被颠覆了？
2. **决策复盘** — 哪个决定最值得重看？
3. **盲区发现** — 昨天忽略了什么？
4. **能量信号** — 能量状态怎样？什么涨什么耗？

## 快速模式

赶时间时输入 `/morning-review quick`，5 分钟完成核心复盘 + 优先级提取。

## 安装

```bash
# 1. 将目录放到 Claude Code skills 目录
cp -r morning-review ~/.claude/skills/

# 2. 在上下文中输入 /morning-review 启动
```

### 依赖

- Claude Code 桌面端（或兼容的 AI 编程工具）
- （可选）工作日志系统，用于提供昨日数据

### 数据源说明

| 数据源 | 权重 | 说明 |
|--------|------|------|
| 工作日志 | 核心（90%） | Claude Code 日报或类似工作记录 |
| 屏幕记录 | 辅助 | Screenpipe 等工具（可选） |
| 笔记工具 | 辅助 | 语音灵感记录（可选） |
| 阅读记录 | 辅助 | 电子书阅读记录（可选） |
